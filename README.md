## How to use:

1. Install the repository 
2. Install required modules using "pip install -r requirements.txt"
3. Run main.py
4. Use **arrows** to rotate cube. Use **left shift** to rotate slower and **left ctrl** to rotate faster.

i. You can also create virtual enviroment (venv), but its not necessary.
